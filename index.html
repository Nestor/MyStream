Welcome

MyStream is an open source streaming backend (along with its own theme), for you to host your own streaming website, or even your own portfolio website to show off your videos!

MyStream is also amazingly powerful giving you a lot of flexibility with a very easy configuration, there is no other script like it (well not free anyway).

Updates to Come

We know that as time goes by, more users will want more features, that's why we've planned a few updates in advanced, well at least the features that are going to come with them:

  • User Login System with Recently Watched
  • Better Design
  • Better 3rd Party Theme Integration
  • Enhanced administrator panel loading (Loads everything in one go so when you switch between pages, the script wont verify you again, meaning you have a quicker loading time | Also means when you switch pages there will be I think no wait time).
  • Views System + Most Watched

Instructions

Well if your eager to get started, you need these instructions:

  1. Create a database and a user account to go with it, or you could use a master account.

  2. Edit the MySQL information in the config/configuration.php file to match. -> $serverip = "localhost";
    $username = "root";
    $password = "root";
    $dbname = "mystream";

  3. Edit the configuration file and put in the rest of the required values.

  4. Edit core/init.php AND ONLY CHANGE THE LINE:
    define(ROOT, $_SERVER["DOCUMENT_ROOT"] . "/Stream"); // Edit the /stream part
    if you place the file in no folder, then get rid of /stream, if you placed it in a folder then do /foldername DO NOT LEAVE / AT THE END!
    Only edit the part /Stream !!!!
